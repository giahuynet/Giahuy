### Markdown

Markdown là một cú pháp nhẹ và dễ sử dụng để tạo kiểu cho bài viết của bạn. Nó bao gồm các quy ước cho

```markdown
Syntax highlighted code block

# Header 1
## Header 2
### Header 3

- Bulleted
- List

1. Numbered
2. List

**Bold** and _Italic_ and `Code` text

[Link](url) and ![Image](src)
```
